# Security Policy

Alcove Dux is local-first and may process sensitive documents. Please report vulnerabilities privately.

## Reporting

Open a private vulnerability report through GitHub Security Advisories:

- <https://github.com/Spitfire-Cowboy/alcove-dux/security/advisories/new>
- <https://github.com/Spitfire-Cowboy/alcove-dux/security/policy>

If GitHub private reporting is unavailable, contact the maintainers through the repository owner.

Please include:

- affected version or commit
- reproduction steps
- expected impact
- whether private documents, paths, embeddings, or reports could be exposed

## Scope

Security-sensitive areas include:

- document ingestion and parsing
- local API file upload
- report export
- vector indexes and embedding caches
- dependency and model supply chain
- hosted deployment guidance

Please omit private documents and personally identifying information from public issues.
