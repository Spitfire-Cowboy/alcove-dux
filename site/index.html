Local-first

The demo runs in the page. The Python package can run locally for PDF, DOCX, Markdown, and plain text review.

Evidence-oriented

Reports focus on spans, offsets, scores, and review notes instead of unsupported conclusions.

Privacy boundary

Public reports avoid local file paths and source text. Local review reports can include more context for private use.
